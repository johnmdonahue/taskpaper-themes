# Taskpaper Themes

A couple [Taskpaper](http://www.hogbaysoftware.com/products/taskpaper) themes.

## <a name="usage"></a>Usage

Copy theme to `~/Library/Application Support/TaskPaper/Themes/`. Restart TaskPaper.
